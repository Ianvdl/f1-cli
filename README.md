# f1-cli
A command line interface wrapper for the Fast-F1 library

```
usage: cli.py [-h] [-y YEAR] [-n NUMBER] [-o OUTPUT] [-r RELOAD]

optional arguments:
  -h, --help            show this help message and exit
  -y YEAR, --year YEAR  Session year
  -n NUMBER, --number NUMBER
                        Session number
  -o OUTPUT, --output OUTPUT
                        Output directory for CSV files and Fast-F1 cache
  -r RELOAD, --reload RELOAD
                        Invalidate the current Fast-F1 cache data
```
